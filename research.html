Research & Projects

Here I write about the research and the projects that I've had the chance to be a part of them.

Artifical images are created using Dall-E 3 through Bing's API

Evaluating Educational Capabilites of Large Language Models

In this research, we carefully studied large language models capability to answer exam questions and grade them, given the correct answer. We implemented variety of prompting strategies and designed experiments to have a rigorous analysis. The target of this project is to facilitate use of AI for teachers and professors.

This work has been supervised by prof. Antoinne Bosselut and prof. Tanja Kaser, both the EPFL university, as a part of my internship. Paper and a link to project resources will be ready soon.

Evaluating 3D Understanding of Large Language Models

In this research, we designed a pipeline to create an artificial dataset called DiceFlip, which is an extention to the CoinFlip dataset. Language model has to determine what specific side of the dice is facing upwards, after certain amount of rolling. Also the effect of chain-of-though prompting and few-shot learning is studied in this work.

This research was supervised by Syrielle Montariol, as a part of my internship at NLP lab, EPFL university, led by prof. Antoinne Bosselut. Links to the project resources will be available soon.

Continous Pretraingin of LLMs on Medical Data

In this reseach, we carefully extracted a high-quality medical corpus and finetuned open access large language models in various sizes on that data. At the next step, we aim to instruction-tune the language model to make it able to perform as a medical chatbot.

This work has been supervised by prof. Antoinne Bosselut and prof. Martin Jaggi, both the EPFL university, as a part of my internship. Here is a link to the original paper.

Robustness Analysis of Computer Vision Models

In this research, we focused on evaluation of the robustness of computer vision models against adversarial attacks. Particulerly, we studied adversarial training and how we can improve it, regarding accuracy and robustness. This reseach is still onging.

This research also has been a part of my bachelorette project. Its supervised by prof. Sajjad Amini.

Deep Learning Course Final Project

For the final project of the deep learning course (Fall 2021) we implemented a depth estimation and an object detection pipeline, using Keras, Tensorflow.

This project was done in collaboration with my friend, Mahyar Jafari. Here is a link to this project.

Machine Course Final Project

For the final project of the machine learning course (Fall 2021) I implemented a feature extraction (both time-domain and frequency-domain) and model selection pipeline for a dataset, consisting of EEG samples. Implemetation was done using the scikit-learn library. The goal was to figure out during the recording the test EEG signal, what specific word was shown to the subject.

Here is a link to this project.

Implemntation was done using Java language and the Swing library for the graphics.
